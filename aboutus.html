米数信息 — 洞察数据，筑梦未来
上海米数信息技术有限公司（MESURE）于2014年12月12日成立，公司初创时其经营活动主要是期货期权交易大数据的挖掘分析、期货大宗商品的 价格行情走势预测、期货交易过程中的波动和异常交易的风控预警，以及期货期权程序化交易策略系统等四个垂直细分的领域，为金融期货机构、交 易员、期货投资者和对冲基金机构等客户提供虽化分析、程序化交易和风控预警的软件和数据挖掘分析等服务。
米数信息充分运用长期积累的信息技术行业研发经验、多年金融期货交易与数据分析市场摸索经验以及对客户市扬的需求分析，致力于将自主研发的 量化指标模型创新技术应用于传统期货交易系统中去，提升与改善现有交易系统的量化和自动化交易的水平，重点是成为提供交易大数据挖掘技术解 决方案的供应商和研究如何通过信息技术量化交易行为、科学理性地投资交易。
随着互联网金融发展，期货交易大数据挖掘分析已成为期货量化交易创新研究的重要领域，目前的大数据挖掘分析主要包括宏观经济运行数据分析挖 掘、期货大宗商品价格波动因素数据模型分析、期货行情交易数据分析、投资者期货交易数据挖掘分析等，其中投资者期货交易数据挖掘分析最为行 业关注。
米数信息依据投资者期货交易数据挖掘分析的市场需求自主独创研发了盈利能力数据分析模型、交易风格数据分析模型、风险控制模型分析、品种收 益数据分析模型和资金运用分析模型的FATA平台系统，通过分布式云计算对海量碎片化交易记录数据逬行高速计箅处理分析，分析投资者交易过程、 量化交易行为和掌握交易习惯；依据期货交易所和期货公司每曰期货品种合约的成交量、多头持仓和空头持仓数据统计分析的FTPA平台系统 ，平台 系统包含了中国金融期货交易所 ( CFFEX )、上海期货交易所 ( SHFE ) 、郑州商品交易所（ ZCE ) 、大连商品交易所 ( DCE ) 和国内185家期 货公司的成交持含数据统计和深度分析。
